# 🤖 AskYourDocs AI

AI chatbot that reads your PDF files and answers questions based on their content using RAG (Retrieval-Augmented Generation).

## 🚀 Features
- Upload PDF files
- Ask questions about your documents
- Accurate answers using AI
- Built with Streamlit + Gemini + FAISS

## 🛠 Tech Stack
- Python
- Streamlit
- Google Gemini API
- LangChain
- FAISS
- HuggingFace Embeddings

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔐 Setup API Key

Create `.streamlit/secrets.toml`:

```
GOOGLE_API_KEY="your_api_key_here"
```
